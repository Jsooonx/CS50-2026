# CS50x 2026 – Harvard University

This repository contains my solutions and projects from **CS50x 2026: Harvard University's Introduction to Computer Science**.

CS50x is Harvard's introduction to the intellectual enterprises of computer science and the art of programming, taught by David J. Malan.

## About This Repository

This repo documents my progress throughout the course, including:

- Weekly problem sets
- Programming exercises
- Final project
- Experiments and improvements beyond the required tasks

The goal is not only to complete the course, but to deeply understand:
- Computational thinking
- Algorithms and efficiency
- Memory management
- Software design principles

## Technologies Used

- C (GCC)
- Python
- SQL (SQLite)
- HTML, CSS, JavaScript
- Flask
- Linux (WSL2 Ubuntu)
- Git & GitHub

## Running the Programs

### C Programs
```bash
gcc program.c -o program
./program
```
### Python Programs
```bash
python3 program.py
```

#### Don't look at the Makefile files or venv folder, that is only for local CS50 library dependencies.
